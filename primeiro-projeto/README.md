Calculadora de consumo elétrico inteligente

💻    Linguagem usada.

Programa utilizado Python



⚡    Fórmula utilizada para o cálculo.

 (potencia x horas dia x 30) / 1000



▶️    Instruções para executar o programa.

1️⃣ Inserir nome do aparelho, 
2️⃣ Potência em watts (W) 
3️⃣ Tempo médio diário em horas
